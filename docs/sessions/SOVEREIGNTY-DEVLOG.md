### W3 ‚Äî `wsf-seal` (seal service, first axum 0.8 HTTP surface) ‚Äî DONE
New crate `crates/wsf-seal` ‚Äî network service over `fabric-envelope`. **Lights up the F4 seal seam:** the per-envelope AES-256-GCM data key is now **wrapped by real OpenBao Transit** (`transit/encrypt|decrypt` ‚Äî Transit does symmetric AEAD; the ML-DSA *signing* decision is untouched).
